MeasureSourcesBuilder& lsst::meas::algorithms::MeasureSourcesBuilder::setCentroider ( CentroidControl const &  centroidControl)

Set the centroid algorithm run before all other algorithms to refine the center point.

The given centroid algorithm will inserted at the front of the list, and if it runs successfully its output will be used as 'center' argument passed to all subsequent algorithms.

This also registers a flag in the schema, 'flags.badcentroid', that will be set if the centroid algorithm did not succeed and hence the center passed to subsequent algorithms was the user's center argument or peak value (depending on which overload of apply is called).
